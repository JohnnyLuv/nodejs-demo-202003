### 运行
```bash
$ yarn serve
```

### 添加环境变量（这个命令是每次重启终端都要执行的）
```bash
$ export PATH=/usr/local/mongodb/bin:$PATH

$ sudo mongod --dbpath=/Users/[yourusername]/mongodb/data/db
```
